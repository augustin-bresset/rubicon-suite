| Feature                             | Path                                                                        |
| ----------------------------------- | --------------------------------------------------------------------------- |
| View list of all models             | PDP Lobby (default view)                                                    |
| Display model image                 | PDP Lobby → click **Model** (upper right)                                   |
| Show model drawing code             | PDP Lobby → click **Drawing** (upper right)                                 |
| List all models                     | PDP Lobby → click **List** next to **Model**                                |
| Open “Make New Product” view        | PDP Lobby → click **New** (bottom menu)                                     |
| Create a blank product              | Make New Product view → click **Make Blank**                                |
| Copy existing product               | Make New Product view → click **Copy**                                      |
| View costing summary                | PDP Lobby → lower pane (default **Costing Summary**)                        |
| Switch to weight summary            | PDP Lobby → click **Weight Summary**                                        |
| View stones details                 | PDP Lobby → click **Stones**                                                |
| View metals details                 | PDP Lobby → click **Metals**                                                |
| View labor cost table               | PDP Lobby → click **Labor etc.**                                            |
| Open parts page                     | Labor etc. pane → click **Parts**                                           |
| View matching suggestions           | PDP Lobby → click **Matching**                                              |
| Manage stone types and categories   | PDP Toolbar → **Manage** → **Stones And Diamonds** → **Types, Shapes etc.** |
| Manage stone shapes, shades & sizes | Stone Info → click **Other Info.**                                          |
| Manage stone setting costs          | Stone Info → click **Settings**                                             |
| Manage stone unit costs             | PDP Toolbar → **Manage** → **Stones And Diamonds** → **Unit Costs**         |
| Print stone cost chart              | Stone Cost Chart view → click **Print**                                     |
| Manage stone unit weights           | PDP Toolbar → **Manage** → **Stones And Diamonds** → **Unit Weights**       |
| Manage metals                       | PDP Toolbar → **Manage** → **Metals**                                       |
| Manage parts                        | PDP Toolbar → **Manage** → **Parts**                                        |
| Manage margins                      | PDP Toolbar → **Manage** → **Margins**                                      |
| View “Misc” margin sub-view         | Manage Margins → click **Misc**                                             |
| View metal margin sub-view          | Manage Margins → click **Metal**                                            |
| View conditional stone margins      | Manage Margins → click **Stone** → **Conditional**                          |
| View normal stone margins           | Manage Margins → click **Stone** → **Normal**                               |
| Manage ornament categories          | PDP Toolbar → **Manage** → **Ornament Categories**                          |
| Manage currencies                   | PDP Toolbar → **Manage** → **Currencies**                                   |
| Tools → Options                     | PDP Toolbar → click **Tools** → **Options**                                 |
| Tools → Reports                     | PDP Toolbar → click **Tools** → **Reports**                                 |
| Tools → Check Data                  | PDP Toolbar → click **Tools** → **Check Data**                              |
