| Feature                                  | Path                                                                                                  |
| ---------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Open SIS Lobby                           | Launch SIS → login → arrive at **Lobby**                                                              |
| View and Edit Parties                    | Lobby → **Maintains** → **Parties**                                                                   |
| Select Party (view only)                 | Maintain Parties → **Select Party** dropdown                                                          |
| Edit General Party Information           | Maintain Parties → **General** tab                                                                    |
| Edit Party Defaults                      | Maintain Parties → **Defaults** tab                                                                   |
| Enable “Is a Vendor” Option              | Defaults tab → tick **Is a Vendor**                                                                   |
| Edit Shipment Information                | Maintain Parties → **Shipment Info** tab                                                              |
| Edit Bank Information                    | Maintain Parties → **Bank Info** tab                                                                  |
| Create New Party                         | Maintain Parties → click **+ (New)** → fill fields → **Save**                                         |
| Open Sales Quotations                    | Lobby → **Maintains** → **Sales** → **Quotations**                                                    |
| Open Sales Orders                        | Lobby → **Maintains** → **Sales** → **Orders**                                                        |
| Open Sales Invoices                      | Lobby → **Maintains** → **Sales** → **Shipment Goods**                                                |
| Edit Order General Fields                | Sales Orders → **General** tab (Doc Name, ID, Closed, Margin, Created, Due Date)                      |
| Edit Order Items – General               | Sales Orders → **Items/General**                                                                      |
| Edit Order Items – Instructions          | Sales Orders → **Items/Instructions**                                                                 |
| Edit Order Items – Sizes                 | Sales Orders → **Items/Sizes**                                                                        |
| Edit Order Items – Weights               | Sales Orders → **Items/Weights**                                                                      |
| Edit Order Items – Profit                | Sales Orders → **Items/Profit**                                                                       |
| Edit Order-Specific Data                 | Sales Orders → **Order** tab (Cust P.O. No., Rcv Mode, Trade Fair, Employee)                          |
| Edit Shipment Details                    | Sales Orders → **Shipment** tab (Ship to, Consignee Bank, For Acc. Of, Book, Page)                    |
| View Child Documents                     | Sales Orders → **Child Documents** tab                                                                |
| View Profit Calculations                 | Sales Orders → bottom section (Profit calculation tables)                                             |
| Create New Sales Document                | Sales Orders → click **New**                                                                          |
| Print Sales Document                     | Sales Orders → click **Print**                                                                        |
| View Product Prices                      | Sales Orders → click **Prices**                                                                       |
| Copy Items from Another Document         | Sales Orders → click **Copy**                                                                         |
| Open Customer Lookup                     | Sales Orders → click **Customers**                                                                    |
| View Metal Requirement Summary           | Sales Orders → click **Metal Req.** (Metal Weight Summary)                                            |
| Maintain Parties via Toolbar             | Sales Orders → Toolbar → **Maintain** → **Parties**                                                   |
| Open Document Browser                    | Sales Orders → Toolbar → **Tools** → **Doc. Browser**                                                 |
| Open Product Browser                     | Sales Orders → Toolbar → **Tools** → **Product Browser**                                              |
| Update Weights for Current Item          | Sales Orders → Toolbar → **Tools** → **Update Weight**                                                |
| Update Weights for All Items             | Sales Orders → Toolbar → **Tools** → **Update Weights (All Items)**                                   |
| Update Costs from PDP                    | Sales Orders → Toolbar → **Tools** → **Update Costs from PDP**                                        |
| Update Cost and Prices from PDP          | Sales Orders → Toolbar → **Tools** → **Update Cost and Prices from PDP**                              |
| Calculate Prices from Raw Material       | Sales Orders → Toolbar → **Tools** → **Calculate Prices from Raw Material** → enter rates → **Start** |
| Exit the Application                     | Lobby → **Application** → **Exit**                                                                    |
| Open Sales Submenu                       | Lobby → **Maintains** → **Sales** → \[Quotations, Orders, Consignment, Returns, Invoices, Repairs]    |
| Open Production Submenu                  | Lobby → **Maintains** → **Production** → \[various document types]                                    |
| View and Edit Misc. Info                 | Lobby → **Maintains** → **Misc. Info.**                                                               |
| View and Edit Business Areas             | Misc. Info. → **Business Areas** sub-tab (Region, Country)                                            |
| View and Edit Trade Fairs                | Misc. Info. → **Trade Fairs** sub-tab (Fair, City, Country, Start, End)                               |
| View and Edit Payment & Shipping Methods | Misc. Info. → **Pay and Ship Methods** sub-tab (Payment Terms, Shipment Methods)                      |
| View and Edit Company Info               | Misc. Info. → **Company Info** sub-tab (Account Info)                                                 |
| Open Reports Menu                        | Lobby → **Reports**                                                                                   |
| View Production Order Forms              | Reports → **Production Order Forms** → \[Assorting, Stone Summary, Casting, Filling, Stock Card]      |
| View Shipment Labels                     | Reports → **Shipment Labels** → \[Big Labels, Small Labels]                                           |
| Generate Other Reports                   | Reports → select report → \[parameters window or direct generation]                                   |
| Configure Report Parameters              | Reports → select → **Parameters** window (filter by doc, customer, stone, category, date, design)     |
| Open Tools Menu                          | Lobby → **Tools** → \[Document Browser, Product Browser, Analysis]                                    |
| Open Analysis Options                    | Tools → **Analysis** → \[Order Analysis, Sales Analysis]                                              |
| Open Help → About                        | Lobby → **Help** → **About**                                                                          |
