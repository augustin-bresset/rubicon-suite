## Description (FR)

Quand on ouvre PDP (Product Definition and Pricing), on arrive sur une page principale qui est le cœur du logiciel :

Barre d’outils en haut :
Application, Manage, Tools, Help.

Sélecteur de modèle : à gauche, un champ permet de choisir un modèle (par ex. R132).
À droite, on voit l’image et éventuellement le dessin technique du modèle sélectionné.

Partie centrale / inférieure : un grand tableau multi-onglets qui affiche différentes vues selon ce qu’on choisit :

Costing Summary : tableau avec les calculs de coût, marge et prix. Les paramètres en haut (marge, pureté, conversion métal, devise, taux USD) influencent le calcul.

Weight Summary : poids des pierres (avant/après recoupe) et poids métal.

Stones : liste des pierres utilisées (LineNum, Setting, RecShape, RecSize, RecWgt, etc.).

Metals : liste des métaux utilisés, avec quantité et pureté.

Labor etc. : main-d’œuvre (gravure, peinture, assemblage…).

Matching : produits assortis (ex. bague + boucles d’oreilles).

Gestion des modèles et références :
En bas, bouton New → permet soit :

Make Blank : créer un nouveau modèle vide.

Copy : dupliquer un modèle existant pour créer une nouvelle référence (variation couleur, métal, pierres).

Menu Manage (dans la barre d’outils) : donne accès à la gestion des bases de données :

Stones and Diamonds : types, formes, tailles, coûts unitaires, poids unitaires.

Metals : coût/once, conversions de poids, puretés.

Parts : composants additionnels.

Margins : marges par type, métal, pierre.

Ornament Categories : gestion des pertes par catégorie.

Currencies : taux de change.

Menu Tools : rapports, export Excel, vérification des données.

Menu Help : informations sur le logiciel.
