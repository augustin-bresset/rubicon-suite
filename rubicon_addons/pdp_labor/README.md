# PDP Labor

CRUD module for labor management in PDP (Product Data Platform).

## Managed tables

- `pdp.labor.type`
- `pdp.labor.cost.model`
- `pdp.labor.cost.product`
- `pdp.addon.type`
- `pdp.addon.cost`

## Main features

- Manage labor types and associated costs (by model and by product).
- Manage addon types and costs.
- Link labor and addons to products and models.
