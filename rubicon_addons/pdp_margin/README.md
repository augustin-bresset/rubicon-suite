# PDP Margin

CRUD module for margin management in PDP (Product Data Platform).

## Managed tables

- `pdp.margin`
- `pdp.margin.addon`
- `pdp.margin.labor`
- `pdp.margin.metal`
- `pdp.margin.part`
- `pdp.margin.stone`
- `pdp.margin.stone.conditional`

## Main features

- Manage margins for addons, labor, metals, parts, and stones.
- Conditional margins for stones by category and cost.
