# PDP Metal

CRUD module for metal and part management in PDP (Product Data Platform).

## Managed tables

- `pdp.metal`
- `pdp.metal.purity`
- `pdp.part`
- `pdp.part.cost`

## Main features

- Manage metals, their purities, and associated costs.
- Manage parts and their costs by purity.
- Link metals and parts to products and models.
