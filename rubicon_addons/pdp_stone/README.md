# PDP Stone

CRUD module for stone management in PDP (Product Data Platform).

## Managed tables

- `pdp.stone`
- `pdp.stone.category`
- `pdp.stone.shade`
- `pdp.stone.shape`
- `pdp.stone.size`
- `pdp.stone.type`
- `pdp.stone.weight`

## Main features

- Create, update, and delete stone records and related categories, shapes, sizes, shades, and types.
- Link stones to products and compositions.
- Manage stone weights and characteristics.
